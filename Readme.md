# Streamlit Data Analytics Projects

A collection of interactive dashboard projects built using Python and Streamlit while practicing data analysis, visualization, and dashboard development.

These projects focus on generating synthetic datasets, performing analysis, building visualizations, and creating interactive dashboards.

---

## Projects

### 1. Student Marks Analytics Dashboard
Features:
- Student performance analysis
- Subject-wise insights
- Top performers
- Correlation heatmaps
- Pivot table analysis

Live Demo:
